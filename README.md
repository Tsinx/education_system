# 重庆交通大学“火山杯”AI 应用创新大赛参赛项目：教师课程 AI 智能体平台

本仓库为重庆交通大学“火山杯”AI 应用创新大赛参赛作品源码，用于展示团队围绕高校教师备课与课程建设场景研发的教育垂直领域 AI Harness 系统。

项目面向学校教师的真实教学准备流程：教师上传课程资料后，系统可自动完成知识抽取、知识库构建、知识图谱组织，以及课程大纲、课堂设计、教案、习题与 PPT 脚本等教学产物生成，并支持知识图谱管理与学习通模板导出。

相较于仅停留在“大模型问答 + 简单生成 + 基础知识检索”阶段的通用工作流产品，本项目进一步强调半自主智能体能力：支持自动知识抽取、按任务调用外部 skill 查询最新信息、结合 Python 解释器校验理工科计算题，并在生成过程中进行反思与修正，以提升教学场景下的准确性、可用性与稳定性。

## 参赛说明

- 赛事名称：重庆交通大学“火山杯”AI 应用创新大赛
- 项目属性：参赛作品，非通用开源模板搬运项目
- 应用方向：AIGC + 教育教学
- 核心定位：面向高校教师备课场景的半自主垂直领域 AI 智能体平台

## 技术栈

- 前端：Vite + React + TypeScript + Ant Design
- 后端：Python + FastAPI + Pydantic
- 数据：SQLite
- 架构：前后端分离，API 统一前缀 `/api/v1`

## 本地启动

### 后端

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000
```

### 前端

```bash
cd frontend
npm install
npm run dev
```

## 当前能力

- 教师工作台可视化流程：上传资料、参数配置、生成任务、结果预览
- 课程资料自动解析、知识抽取与知识图谱构建
- 结合外部 skill 查询最新信息，增强教学内容时效性
- 借助 Python 解释器校验理工科计算过程与结果，提高题目生成可靠性
- 支持生成结果的反思、修正与多轮优化
- 知识图谱树状展示与节点删除（递归删除/子节点提升）
- 知识图谱导出学习通模板（xlsx）
- API 不可用时前端自动回退演示数据，保证演示稳定性
