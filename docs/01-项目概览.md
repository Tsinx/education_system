# 项目概览

## 项目名称

教师课程 AI 智能体平台（education_system）

## 目标

- 教师上传课程资料后，自动抽取章节和知识点。
- 构建课程知识图谱并支持人工修订。
- 生成教学相关产出（教案、题目、PPT 脚本等）。
- 提供 RAG 查询能力，支撑教学问答和内容检索。

## 当前技术栈

- 前端：Vite + React + TypeScript + Ant Design
- 后端：FastAPI + Pydantic
- 数据：SQLite（课程、资料、知识点、边关系、AI 结果）
- 模型层：DashScope / OpenRouter / DeepSeek + 本地 embedding/rerank（可切换）

## 业务主流程

1. 创建课程。
2. 上传资料并绑定课程。
3. 自动提取章节与知识点。
4. 知识图谱可视化、删除与重排。
5. 导出学习通知识图谱模板。
6. 启动 AI 产出任务与结果导出。
