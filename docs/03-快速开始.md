# 快速开始

## 环境要求

- Node.js 20+
- Python 3.13+
- Windows / macOS / Linux（当前项目在 Windows 路径下开发）

## 1. 启动后端

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000
```

后端地址：`http://localhost:8000`

## 2. 启动前端

```bash
cd frontend
npm install
npm run dev
```

前端地址：`http://localhost:5173`

## 3. 接口文档

- Swagger UI：`http://localhost:8000/docs`
- ReDoc：`http://localhost:8000/redoc`

## 4. 常用开发命令

### 前端

```bash
cd frontend
npm run dev
npm run build
npm run lint
```

### 后端

```bash
cd backend
python -m py_compile app\main.py
```

## 5. 常见问题

### 依赖安装慢

- Python 侧建议先升级 `pip`。
- 前端建议使用稳定网络或企业镜像。

### 跨域问题

- 在 `backend/app/core/config.py` 中调整 `cors_origins`。

### 导出模板不生效

- 确认文件存在：`backend/app/assets/importTopicTemplate.xlsx`。
- 确认导出接口返回 200 且浏览器已下载 `.xlsx` 文件。
