# 前端说明

## 技术栈

- React 19 + TypeScript
- Vite 8
- Ant Design 6
- Axios + React Router

## 页面路由

- `/dashboard`：工作台
- `/courses/:courseId`：课程详情
- `/courses/:courseId/knowledge-graph`：知识图谱
- `/settings`：设置
- `/profile`：个人中心

## 代码组织

- `src/layouts/`：页面骨架。
- `src/pages/`：业务页面。
- `src/api/client.ts`：API 封装。
- `src/types/`：类型定义。
- `src/data/mock.ts`：演示兜底数据。

## 知识图谱页面

页面文件：`src/pages/KnowledgeGraphPage.tsx`

已实现：

- 树状层级展示
- 删除节点（支持递归删除或子节点提升）
- 导出学习通模板按钮

## API 调用约定

- 所有请求默认基于 `http://localhost:8000/api/v1`。
- 文件导出请求需使用 `responseType: 'blob'`。
- 对导出接口，从 `content-disposition` 解析下载文件名。

## 前端质量建议

- 合并前执行 `npm run build` 和 `npm run lint`。
- 对关键页面（知识图谱、课程详情）优先做手动回归。
- 新增接口时先补充 `types` 类型，再在 `client.ts` 增加方法。
