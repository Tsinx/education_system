# 知识图谱导入导出规范

## 目标

将课程知识点导出为学习通知识图谱可直接导入的 Excel 文件。

## 导出模板

- 模板文件：`backend/app/assets/importTopicTemplate.xlsx`
- 导出接口：`GET /api/v1/knowledge/courses/{course_id}/graph-export`

## 工作表与行列约定

- 使用 `Sheet3`（若不存在则回退首个工作表）。
- 第 1 行和第 2 行是模板说明与表头，必须保留。
- 数据从第 3 行开始写入。

列约定（A~O）：

- `A~G`：知识点层级列（每行只填一列）
- `H`：前置知识点（英文分号 `;` 连接）
- `I`：后置知识点（英文分号 `;` 连接）
- `J`：关联知识点（英文分号 `;` 连接）
- `K~N`：扩展字段（当前可留空）
- `O`：知识点说明

## 层级映射规则

学习通以“章”为最高层，因此导出时课程内知识点层级整体后移一级：

- 导出列号 = `min(7, max(1, level + 1))`

示例：

- 原 `L1` -> 导出到 `B` 列
- 原 `L2` -> 导出到 `C` 列

这样可保证“章”为顶层时，章内一级知识点是其子级，而不是同级。

## 删除节点与层级一致性

后端删除节点支持：

- `delete_descendants=true`：递归删除所有后代。
- `delete_descendants=false`：保留后代并提升一级。

该策略可保证导出前后层级结构连续。

## 前端下载行为

前端按钮调用 `exportCourseKnowledgeGraph(courseId)`：

- 以 `blob` 接收数据。
- 从 `content-disposition` 提取 UTF-8 文件名。
- 创建临时 `<a download>` 触发下载。

## 验收清单

1. 导出文件可被学习通成功导入。
2. 章内一级知识点不是与章同级。
3. 前置/后置/关联在导出列中无乱码、分隔符正确。
4. 模板头两行保持不变。
