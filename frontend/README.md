# Frontend（React + TypeScript）

## 启动

```bash
cd frontend
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 代码结构

- `src/pages/`：业务页面
- `src/layouts/`：布局
- `src/api/client.ts`：接口调用
- `src/types/`：类型定义
- `src/data/mock.ts`：演示兜底数据

## 关键页面

- `CourseDetailPage`：课程详情与素材管理
- `KnowledgeGraphPage`：知识图谱层级展示、节点删除、学习通模板导出

更多请查看项目文档：`../docs/README.md`
